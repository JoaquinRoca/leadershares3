# README #

## Development ##

Leadershares (leadershares3) runs on Ruby 2.0.0 (see .ruby-version).

To get Leadershares up and running, do the following:

1. Set up config/database.yml (see example file).

2. Set up config/application.yml (see example file).

3. Make sure the `bundler` is installed and run `bundle` in the terminal.

4. The database used in development mode is sqlite3. Run `rake db:create`.

5. Migrate the database: `rake db:migrate`.
